# Frontend

The user-facing layer: React + TypeScript + custom SVG canvas + Tailwind CSS.

## Contents

- **pages/** — Per-page specifications covering layout, components, data flow, and interactions.
- **ux-principles/** — Cross-cutting design principles and reusable UX patterns.
- **deployment/** — Docker setup, environment variables, and networking.

Read `ux-principles/` before working on any page, then read the relevant page spec in `pages/`.
