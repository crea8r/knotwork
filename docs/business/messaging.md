# Messaging

## Core Message

Run complex work with agents, SOPs, and human review.

## Supporting Message

Knotwork turns SOPs, project context, and human decisions into repeatable execution.

## Message Variants To Test

### Variant 1

Hook:

Run complex work with agents, SOPs, and human review.

Main benefit:

Knotwork turns your handbook into execution. Agents follow documented process, pause for people when needed, and improve over time instead of becoming another black box.

Trust support:

- visual workflows
- task and project tracking
- run history and audit trail
- human approvals and escalations
- external agent execution

### Variant 2

Hook:

Your team already has a way of working. Knotwork makes it executable.

Main benefit:

Most AI tools automate tasks. Knotwork operationalizes judgment-heavy work by combining handbook knowledge, project context, and structured human intervention.

Trust support:

- reusable workflows
- project and task structure
- traceable agent runs
- escalation handling
- collaboration around work

### Variant 3

Hook:

Not every workflow should be fully automated.

Main benefit:

Knotwork is built for work that needs judgment. Agents do the repeatable parts, humans step in at the right moments, and every decision stays tied to the knowledge used.

Trust support:

- workflow orchestration
- project and task organization
- agent-compatible execution
- visibility into each run
- resumable work after review
